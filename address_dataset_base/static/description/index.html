Frequently Asked Questions

Quick answers to the most common setup and operational questions.

Does this base module include country datasets by default?

No. This base module provides shared models and logic. Country modules add dataset records (TR, DE, US, ES, etc.).

Can I edit postal records manually?

Yes. The module is installed as an app and users can create, edit and manage records from the postal menu.

Are latitude and longitude fields optional?

Yes. Coordinates are supported and can also be hidden from configuration when needed.

Is it compatible with Community and Enterprise?

Yes. The module is prepared for Odoo Community, Enterprise (on-premise), and Odoo.sh for versions 11.0, 12.0, 13.0, 14.0, 15.0, 16.0, 17.0, 18.0, and 19.0.

Can partner addresses auto-fill from neighborhood selection?

Yes. Selecting neighborhood/postal scope can auto-populate related address fields on partners.

How do I request technical support?

Send your environment details, Odoo version, and issue steps to support@algoritmetic.com. Including screenshots and expected behavior helps speed up resolution.
